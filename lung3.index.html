2. Lung Diseases – أمراض الرئة

❤️‍🔥 أمراض انسدادية / Obstructive Diseases

🔹 Chronic Obstructive Pulmonary Disease (COPD) – مرض الانسداد الرئوي المزمن

😤 Symptoms / الأعراض:

  1. Shortness of breath / ضيق تنفس
  2. Chronic cough / سعال مزمن
  3. Excessive mucus production / إفراز البلغم بكثرة
  4. Wheezing / صفير عند التنفس

🧠 Cause / السبب:

  1. Smoking / التدخين
  2. Long-term exposure to lung irritants / التعرض الطويل للمهيجات الرئوية
  3. Genetic factors (alpha-1 antitrypsin deficiency) / عوامل وراثية

🔬 Diagnosis / التشخيص:

  1. Spirometry / قياس التنفس
  2. Chest X-ray / أشعة الصدر
  3. CT scan / التصوير المقطعي

💊Treatment / العلاج:

  1. Bronchodilators / موسعات الشعب الهوائية
  2. Inhaled corticosteroids / الكورتيكوستيرويدات المستنشقة
  3. Oxygen therapy / العلاج بالأكسجين
  4. Pulmonary rehabilitation / إعادة تأهيل الرئة

⚠️ Complications / المضاعفات:

  1. Respiratory infections / التهابات الجهاز التنفسي
  2. Heart problems / مشاكل القلب
  3. Lung cancer / سرطان الرئة
  4. Pulmonary hypertension / ارتفاع ضغط الدم الرئوي

🔹 Emphysema – النفاخ الرئوي

😤 Symptoms / الأعراض:

  1. Shortness of breath / ضيق التنفس
  2. Reduced ability for physical activity / انخفاض القدرة على النشاط البدني
  3. Barrel chest / صدر برميلي
  4. Wheezing / صفير

🧠 Cause / السبب:

  1. Smoking / التدخين
  2. Alpha-1 antitrypsin deficiency / نقص ألفا-1 أنتيتريبسين
  3. Long-term exposure to air pollutants / التعرض الطويل لتلوث الهواء

🔬 Diagnosis / التشخيص:

  1. Pulmonary function tests / اختبارات وظائف الرئة
  2. Chest X-ray / أشعة الصدر
  3. CT scan / التصوير المقطعي

💊Treatment / العلاج:

  1. Bronchodilators / موسعات الشعب الهوائية
  2. Inhaled steroids / الستيرويدات المستنشقة
  3. Oxygen therapy / العلاج بالأكسجين
  4. Pulmonary rehabilitation / إعادة تأهيل الرئة

⚠️ Complications / المضاعفات:

  1. Respiratory failure / الفشل التنفسي
  2. Pneumothorax / استرواح الصدر
  3. Cor pulmonale / القلب الرئوي

🔹 Chronic Bronchitis – التهاب الشعب الهوائية المزمن

😤 Symptoms / الأعراض:

  1. Productive cough for at least 3 months / سعال منتج لمدة 3 أشهر على الأقل
  2. Shortness of breath / ضيق التنفس
  3. Wheezing / صفير
  4. Chest tightness / ضيق في الصدر

🧠 Cause / السبب:

  1. Smoking / التدخين
  2. Air pollution / تلوث الهواء
  3. Occupational exposures / التعرض المهني

🔬 Diagnosis / التشخيص:

  1. Clinical symptoms / الأعراض السريرية
  2. Chest X-ray / أشعة الصدر
  3. Spirometry / قياس التنفس

💊Treatment / العلاج:

  1. Bronchodilators / موسعات الشعب الهوائية
  2. Steroids / الستيرويدات
  3. Oxygen therapy / العلاج بالأكسجين
  4. Antibiotics for infections / المضادات الحيوية للعدوى

⚠️ Complications / المضاعفات:

  1. Recurrent respiratory infections / التهابات تنفسية متكررة
  2. Respiratory failure / الفشل التنفسي
  3. Cor pulmonale / القلب الرئوي

🔹 Asthma – الربو

😤 Symptoms / الأعراض:

  1. Wheezing / صفير
  2. Coughing / سعال
  3. Chest tightness / ضيق في الصدر
  4. Shortness of breath / ضيق التنفس

🧠 Cause / السبب:

  1. Genetic predisposition / الاستعداد الوراثي
  2. Environmental factors / العوامل البيئية
  3. Allergens / المواد المثيرة للحساسية

🔬 Diagnosis / التشخيص:

  1. Spirometry / قياس التنفس
  2. Peak flow measurement / قياس ذروة الجريان
  3. Allergy testing / اختبار الحساسية

💊Treatment / العلاج:

  1. Inhaled corticosteroids / الكورتيكوستيرويدات المستنشقة
  2. Bronchodilators / موسعات الشعب الهوائية
  3. Leukotriene modifiers / معدلات الليكوترين
  4. Avoiding triggers / تجنب المحفزات

⚠️ Complications / المضاعفات:

  1. Severe asthma attacks / نوبات ربو شديدة
  2. Sleep disturbances / اضطرابات النوم
  3. Permanent airway changes / تغيرات دائمة في الشعب الهوائية

🔹 Bronchiectasis – توسع القصبات

😤 Symptoms / الأعراض:

  1. Chronic cough with large amounts of sputum / سعال مزمن مع كميات كبيرة من البلغم
  2. Shortness of breath / ضيق التنفس
  3. Chest pain / ألم الصدر
  4. Recurrent respiratory infections / التهابات تنفسية متكررة

🧠 Cause / السبب:

  1. Respiratory infections / الالتهابات التنفسية
  2. Cystic fibrosis / التليف الكيسي
  3. Immune system problems / مشاكل الجهاز المناعي

🔬 Diagnosis / التشخيص:

  1. CT scan / التصوير المقطعي
  2. Chest X-ray / أشعة الصدر
  3. Sputum culture / زراعة البلغم

💊Treatment / العلاج:

  1. Antibiotics for infections / المضادات الحيوية للعدوى
  2. Bronchodilators / موسعات الشعب الهوائية
  3. Chest physiotherapy / العلاج الطبيعي للصدر
  4. Oxygen therapy / العلاج بالأكسجين

⚠️ Complications / المضاعفات:

  1. Respiratory failure / الفشل التنفسي
  2. Heart failure / فشل القلب
  3. Recurrent pneumonia / الالتهاب الرئوي المتكرر

❤️‍🔥 أمراض تقييدية / Restrictive Diseases

🔹 Idiopathic Pulmonary Fibrosis (IPF) – التليف الرئوي مجهول السبب

😤 Symptoms / الأعراض:

  1. Shortness of breath / ضيق التنفس
  2. Dry cough / سعال جاف
  3. Fatigue / التعب
  4. Unexplained weight loss / فقدان الوزن غير المبرر

🧠 Cause / السبب:

  1. Unknown cause / سبب غير معروف
  2. Genetic factors / عوامل وراثية
  3. Environmental exposures / التعرض البيئي

🔬 Diagnosis / التشخيص:

  1. High-resolution CT scan / التصوير المقطعي عالي الدقة
  2. Pulmonary function tests / اختبارات وظائف الرئة
  3. Lung biopsy / خزعة الرئة

💊Treatment / العلاج:

  1. Antifibrotic medications / الأدوية المضادة للتليف
  2. Oxygen therapy / العلاج بالأكسجين
  3. Pulmonary rehabilitation / إعادة تأهيل الرئة
  4. Lung transplant / زراعة الرئة

⚠️ Complications / المضاعفات:

  1. Respiratory failure / الفشل التنفسي
  2. Pulmonary hypertension / ارتفاع ضغط الدم الرئوي
  3. Lung cancer / سرطان الرئة

🔹 Sarcoidosis – الساركويد

😤 Symptoms / الأعراض:

  1. Shortness of breath / ضيق التنفس
  2. Persistent dry cough / سعال جاف مستمر
  3. Fatigue / التعب
  4. Swollen lymph nodes / تورم الغدد الليمفاوية

🧠 Cause / السبب:

  1. Unknown / غير معروف
  2. Immune system overreaction / فرط تفاعل الجهاز المناعي
  3. Genetic predisposition / الاستعداد الوراثي

🔬 Diagnosis / التشخيص:

  1. Chest X-ray / أشعة الصدر
  2. CT scan / التصوير المقطعي
  3. Biopsy / الخزعة
  4. Blood tests / فحوصات الدم

💊Treatment / العلاج:

  1. Corticosteroids / الكورتيكوستيرويدات
  2. Immunosuppressants / الأدوية المثبطة للمناعة
  3. Monitoring / المراقبة

⚠️ Complications / المضاعفات:

  1. Permanent lung damage / تلف دائم في الرئة
  2. Eye inflammation / التهاب العين
  3. Heart problems / مشاكل القلب

❤️‍🔥 أمراض معدية / Infectious Diseases

🔹 Pneumonia – الالتهاب الرئوي

😤 Symptoms / الأعراض:

  1. Cough with phlegm / سعال مع بلغم
  2. Fever / حمى
  3. Shortness of breath / ضيق التنفس
  4. Chest pain / ألم الصدر

🧠 Cause / السبب:

  1. Bacterial infection / العدوى البكتيرية
  2. Viral infection / العدوى الفيروسية
  3. Fungal infection / العدوى الفطرية

🔬 Diagnosis / التشخيص:

  1. Chest X-ray / أشعة الصدر
  2. Blood tests / فحوصات الدم
  3. Sputum culture / زراعة البلغم

💊Treatment / العلاج:

  1. Antibiotics / المضادات الحيوية
  2. Antiviral medications / الأدوية المضادة للفيروسات
  3. Fever reducers / خافضات الحرارة
  4. Oxygen therapy / العلاج بالأكسجين

⚠️ Complications / المضاعفات:

  1. Respiratory failure / الفشل التنفسي
  2. Sepsis / تسمم الدم
  3. Lung abscess / خراج الرئة

🔹 Tuberculosis (TB) – السل

😤 Symptoms / الأعراض:

  1. Persistent cough / سعال مستمر
  2. Chest pain / ألم الصدر
  3. Coughing up blood / سعال الدم
  4. Fever and night sweats / حمى وتعرق ليلي

🧠 Cause / السبب:

  1. Mycobacterium tuberculosis / بكتيريا المتفطرة السلية
  2. Airborne transmission / الانتقال عبر الهواء

🔬 Diagnosis / التشخيص:

  1. Skin test / اختبار الجلد
  2. Blood tests / فحوصات الدم
  3. Chest X-ray / أشعة الصدر
  4. Sputum tests / فحوصات البلغم

💊Treatment / العلاج:

  1. Antibiotics for several months / المضادات الحيوية لعدة أشهر
  2. Directly observed therapy / العلاج تحت الإشراف المباشر

⚠️ Complications / المضاعفات:

  1. Lung damage / تلف الرئة
  2. Spinal pain / ألم العمود الفقري
  3. Joint damage / تلف المفاصل

🔹 Lung Abscess – خراج الرئة

😤 Symptoms / الأعراض:

  1. Fever / حمى
  2. Cough with foul-smelling sputum / سعال مع بلغم كريه الرائحة
  3. Night sweats / تعرق ليلي
  4. Weight loss / فقدان الوزن

🧠 Cause / السبب:

  1. Bacterial infection / عدوى بكتيرية
  2. Aspiration / الشفط
  3. Obstruction / انسداد

🔬 Diagnosis / التشخيص:

  1. Chest X-ray / أشعة الصدر
  2. CT scan / التصوير المقطعي
  3. Sputum culture / زراعة البلغم

💊Treatment / العلاج:

  1. Antibiotics / المضادات الحيوية
  2. Drainage procedures / إجراءات التصريف
  3. Surgery in severe cases / الجراحة في الحالات الشديدة

⚠️ Complications / المضاعفات:

  1. Empyema / الدبيلة
  2. Bronchopleural fistula / ناسور قصبي جنبي
  3. Sepsis / تسمم الدم

🔹 Empyema – الدبيلة

😤 Symptoms / الأعراض:

  1. Fever / حمى
  2. Chest pain / ألم الصدر
  3. Shortness of breath / ضيق التنفس
  4. Cough / سعال

🧠 Cause / السبب:

  1. Pneumonia / الالتهاب الرئوي
  2. Lung abscess / خراج الرئة
  3. Thoracic surgery / جراحة الصدر

🔬 Diagnosis / التشخيص:

  1. Chest X-ray / أشعة الصدر
  2. CT scan / التصوير المقطعي
  3. Thoracentesis / بزل الصدر

💊Treatment / العلاج:

  1. Antibiotics / المضادات الحيوية
  2. Chest tube drainage / تصريف بأنبوب الصدر
  3. Surgical decortication / تقشير جراحي

⚠️ Complications / المضاعفات:

  1. Sepsis / تسمم الدم
  2. Respiratory failure / الفشل التنفسي

  5. Unexplained weight loss / فقدان الوزن غير المبرر

❤️‍🔥 أمراض الأوعية الدموية الرئوية / Pulmonary Vascular Diseases

🔹 Pulmonary Embolism – الانصمام الرئوي

😤 Symptoms / الأعراض:

  1. Sudden shortness of breath / ضيق تنفس مفاجئ
  2. Chest pain / ألم الصدر
  3. Coughing up blood / سعال الدم
  4. Rapid heart rate / سرعة ضربات القلب

🧠 Cause / السبب:

  1. Blood clots from deep vein thrombosis / جلطات الدم من تجلط الأوردة العميقة
  2. Immobility / عدم الحركة
  3. Surgery / الجراحة

🔬 Diagnosis / التشخيص:

  1. CT pulmonary angiography / التصوير المقطعي للشرايين الرئوية
  2. Ventilation-perfusion scan / مسح التهوية-التروية
  3. D-dimer blood test / اختبار الدم للد-دايمر

💊Treatment / العلاج:

  1. Anticoagulants / مضادات التخثر
  2. Thrombolytics / مذيبات الجلطة
  3. Embolectomy / استئصال الصمة

⚠️ Complications / المضاعفات:

  1. Pulmonary hypertension / ارتفاع ضغط الدم الرئوي
  2. Heart failure / فشل القلب
  3. Recurrent embolism / الانصمام المتكرر

🔹 Pulmonary Hypertension – ارتفاع ضغط الدم الرئوي

😤 Symptoms / الأعراض:

  1. Shortness of breath / ضيق التنفس
  2. Fatigue / التعب
  3. Chest pain / ألم الصدر
  4. Swelling in ankles and legs / تورم الكاحلين والساقين

🧠 Cause / السبب:

  1. Idiopathic / مجهول السبب
  2. Genetic factors / عوامل وراثية
  3. Underlying heart or lung disease / أمراض القلب أو الرئة الكامنة

🔬 Diagnosis / التشخيص:

  1. Echocardiogram / تخطيط صدى القلب
  2. Right heart catheterization / قسطرة القلب الأيمن
  3. Pulmonary function tests / اختبارات وظائف الرئة

💊Treatment / العلاج:

  1. Vasodilators / موسعات الأوعية
  2. Anticoagulants / مضادات التخثر
  3. Diuretics / مدرات البول
  4. Oxygen therapy / العلاج بالأكسجين

⚠️ Complications / المضاعفات:

  1. Right heart failure / فشل القلب الأيمن
  2. Arrhythmias / عدم انتظام ضربات القلب
  3. Bleeding in lungs / النزيف في الرئتين

  2. Rapid breathing / تنفس سريع
  3. Low blood oxygen / انخفاض أكسجين الدم

🧠 Cause / السبب:

  1. Sepsis / الإنتان
  2. Pneumonia / الالتهاب الرئوي
  3. Trauma / الصدمة
  4. Aspiration / الشفط

🔬 Diagnosis / التشخيص:

  1. Chest X-ray / أشعة الصدر
  2. Blood tests / فحوصات الدم
  3. CT scan / التصوير المقطعي

💊Treatment / العلاج:

  1. Mechanical ventilation / التهوية الميكانيكية
  2. Oxygen therapy / العلاج بالأكسجين
  3. Treating underlying cause / علاج السبب الأساسي

⚠️ Complications / المضاعفات:
